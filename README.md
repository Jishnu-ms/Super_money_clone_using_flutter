## ⚙️ Getting Started

### Prerequisites
- Flutter SDK
- Python 3.10+
- pip

### Run Flutter App
```bash
flutter pub get
flutter run
```

### Run FastAPI Backend
```bash
cd backend/
pip install -r requirements.txt
uvicorn main:app --reload
```

## 🌐 Deployment

### Flutter Web
```bash
flutter build web
```
Then upload `/build/web` to:
- GitHub Pages
- Firebase Hosting

### FastAPI Backend
Deploy to:
- Render
- Railway
- Deta
